Selectors

Type selectors

Выбор елементов по тегу

                span { 
                      background-color: red; 
                      color: #ffffff; 
                }

ID selectors

Выбор елементов по ID

                span#identified {
                  background-color: red; 
                }

ID selectors

What makes attributes of type ID special is that no two such attributes can have the same value in a conformant document, regardless of the type of the elements that carry them;

Class selectors

Выбор елементов по классу

                span.classy {
                  background-color: red; 
                }

Universal selectors

Выбор всех элементов

                * {
                  background-color: red; 
                }

Attribute selectors

Выбор элементов с определенным атрибутом

                 /* Все span с атрибутом lang */
                span[lang] {
                  font-weight:bold;
                }

Attribute selectors

Выбор элементов с определенным атрибутом

                /* Все span c атрибутом lang="ru" */
                span[lang="ru"] {
                  color:green;
                }

Вес селекторов

A selector's specificity is calculated as follows:
count 1 if the declaration is from is a 'style' attribute rather than a rule with a selector, 0 otherwise ...
count the number of ID attributes in the selector
count the number of other attributes and pseudo-classes in the selector
count the number of element names and pseudo-elements in the selector

Pseudo-elements

Pseudo-elements — Синтаксис

Правила которые создают псевдо-элементы

                selector:pseudo-element {
                  property: value;
                }

Pseudo-elements

  • :after — создает псевдо-элемент после элемента
  • :before — создает псевдо-элемент до элемента
  • ::first-letter — создает псевдо-элемент вокруг первой буквы элемента
  • ::first-line — создает псевдо-элемент вокруг первой строчки элемента
  • ::selection — создает псевдо-элемент вокруг выделенного текста

Pseudo-elements

:: или :?

:: — часть спецификации CSS и попытка разделить псевдо-классы и псевдо элементы. Большинство браузеров поддерживает оба варианта

Css правила

Css правила

                padding: 0; /* Короткая запись */
                 
                padding: 1px 2px 3px 4px; /* Короткая запись */
                 
                /* Полная запись */
                padding-top:    1px;
                padding-right:  2px;
                padding-bottom: 3px;
                padding-left:   4px;

Css правила

                font: bold 12px/14px sans-serif;  /* Короткая запись */
                 
                /* Полная запись */
                font-weight: bold;
                font-size:   12px;
                line-height: 14px;
                font-family: sans-serif;

Css правила

                background: url(img.png) #888 repeat-y top left fixed;
                 
                /* Полная запись */
                background-image:      url(img.png);
                background-color:      #888;
                background-repeat:     repeat-y;
                background-position:   top left;
                background-attachment: fixed;

Css правила

                -webkit-border-radius: 10px;
                 -khtml-border-radius: 10px;
                   -moz-border-radius: 10px;
                    -ms-border-radius: 10px;
                     -o-border-radius: 10px;
                        border-radius: 10px;

Css правила — значения

                width:     100%;             /* px, pt, em */
                font-size: 100%;             /* px, pt, em */
                font-family: Courier, "Lucida Console", monospace;
                 
                color: red;                  /* keyword */
                color: #f00;                 /* #ff0000 hex */
                color: rgb(255, 0, 0 );      /* rgb */
                color: rgba(255, 0, 0, 0.5); /* rgb with alfa */
                color: hsl(0, 100, 100);     /* hsl */

Css правила — значения

                background-image:    url(img.png);  /* url */
                background-image:    linear-gradient( 45deg, blue, red );
                 
                transition-property: opacity, left, top, height;

MDN: At-rule

@import — Синтаксис

Добавление правил из внешнего файла.

                @import url; 
                или
                @import url list-of-media-queries;

@import — Примеры

                        @import url("fineprint.css") print;
                        @import url("bluish.css") projection, tv;
                        @import 'custom.css';
                        @import "common.css" screen, projection;
                        @import url('test.css') screen and (orientation:landscape);

@media — Синтаксис

Группа правил, которая выполняется на определенном типе устройств

                    @media media-types list-of-media-queries {
                     /* media-specific rules */
                    }

media types — all, print, projection, screen

@media — Примеры

                @media screen {
                  body { font-size: 10pt }
                }
                @media screen, print {
                  body { font-size: 13px }
                }

@media — CSS media queries

                @media screen and (max-width: 960) { ... }
                 
                @media all and (orientation: portrait) { ... }
